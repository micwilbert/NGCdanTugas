Belajar tentang environment di data science : set up VSCode, Git Bash, Miniconda
Basic Commands : 
echo : seperti print, text or string 
pwd : memberitahu path sekarang
cd : menuju ke suatu directory
cd ~ : langsung ke root
cd .. : ke user
ls : liat isi
cp : copy
mv : move
mkdir : make directory
rm : remove
rmdir : remove directory

Set Up Conda
conda --version
conda create --name <environment> "python=<seri mana>": bikin environment
conda info --envs : cek environment
conda list : cek program
